# vue3-music

学习vue3，开发一款媲美移动端的音乐项目。

## 项目亮点

- 配合better-scroll开发slider、scroll组件
- v-loading自定义组件开发

## commit规范

用于说明 commit 的类别，只允许使用下面7个标识。

- feat：新功能（feature）

- fix：修补bug

- docs：文档（documentation）

- style： 格式（不影响代码运行的变动）

- refactor：重构（即不是新增功能，也不是修改bug的代码变动）

- test：增加测试

- chore：构建过程或辅助工具的变动
